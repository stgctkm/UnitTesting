# 再代入を避ける


Rentalクラスの料金計算メソッドの再代入を除外します

```java
public double amount() {
    double thisAmount = 0;
    // 一行ごとに金額を計算
    switch (getMovie().getPriceType()) {
        case REGULAR:
            thisAmount += 2;
            if (getDaysRented() > 2)
                thisAmount += (getDaysRented() - 2) * 1.5;
            break;
        case NEW_RELEASE:
            thisAmount += getDaysRented() * 3;
            break;
        case CHILDREN:
            thisAmount += 1.5;
            if (getDaysRented() > 3)
                thisAmount += (getDaysRented() - 3) * 1.5;
            break;
    }
    return thisAmount;
}
```

```java
public double amount() {
    // 一行ごとに金額を計算
    switch (getMovie().getPriceType()) {
        case REGULAR:
            if (getDaysRented() <= 2) return 2;
            return 2 + (getDaysRented() - 2) * 1.5;
        case NEW_RELEASE:
            return getDaysRented() * 3;
        case CHILDREN:
            if (getDaysRented() <= 3) return 1.5;
            return 1.5 + (getDaysRented() - 3) * 1.5;
        default: return 0;
    }
}
```
